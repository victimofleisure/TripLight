*** TripLight source distribution ***

REQUIREMENTS:

TripLight is compatible with the following versions of Visual Studio:
MFC 6.0, VS2005, VS2008, VS2010, and VS2012. To open the project in MFC
6.0 and VS2005, open the .dsw (workspace) file; for VS2008 and later
versions, open the .sln (solution) file. Depending on your version, the
the project conversion wizard may run.

INSTALLATION:

The source distribution of TripLight doesn't have an installer.

UNINSTALLING:

To uninstall TripLight, do the following:

1.  Delete TripLight's program files from the folder you copied them to.
2.  If you ran the non-portable version of TripLight, delete TripLight's
registry entries, using regedit.  The key to delete is:

"HKEY_CURRENT_USER\Software\Anal Software\TripLight"

NOTES:

The binary version of this release is available from the TripLight web
site, http://TripLight.sourceforge.net/, and also from GitHub.
